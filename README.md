# toys_api

## GET START

clone the repository to your directory


```
cd /path_to_folder

git clone https://github.com/kultmet/toys_api.git
```

install [docker](https://docs.docker.com/engine/install/) and [docker-compose](https://docs.docker.com/compose/install/linux/)

start app

```
# command line

docker-compose up -d

```
